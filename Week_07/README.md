### Trie(字典树)
- 字典树的数据结构
    字典树（Trie），又称单词查找树或者键树，是一种树形结构。  
    典型应用于统计和排序大量字符串（但不仅限于字符串），经常被用于搜索引擎统计文本词频。  
    优点：最大限度的减少无谓的字符串的比较，比哈希表查询效率高。     
每个节点都要访问一次
- 每个节点仅仅要访问一次 
- 节点访问顺序：
    - 深度优先：depth first search
    - 广度优先：breadth first search
